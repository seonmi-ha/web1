    8/9 HTML


작심 40시간을 듣다가 무언가 이상한점들이 있어서 유투브로 돌아가 살펴보니, 2013년도에는 서브라임텍스트를 주로 이용했는데, 그때의 버전과 지금의 버전이 달라 훨씬 더 어렵고 복잡하게 느껴졌던 것이었다. 그래서 2017년도에 나온 생활코딩web1-html&internet을 찾고 다시 처음부터 수업을 듣게 되었다.

web1-html&internet은 초보자를 위한 기초 영상으로 작심40시간과 컨텐츠는 비슷하나 하나당 5~10분 내외의 영상이 35개로 이루어져있고, 아가자기한 일러스트를 넣어 현대 컨텐츠 소비시장을 저격하여 새로 만들어진 것 같았다.




-오늘 배운 것 한줄 정리:ATOM을 이용해서 만들기 시작

      1. strong /strong : 굵게
      2. u /u : 밑줄
      3. h1 /h1~h6 /h6 : 글씨크기1>6
      4. br : 한줄띄기
      5. p /p : 단락
      6. p /pstyle="magin-top45px;" : 45픽셀만큼 띄우기
